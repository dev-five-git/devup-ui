# Typography

Devup UI provides a powerful typography system that supports static styles, responsive typography, and theme integration.

## Basic Typography

Define typography styles in `devup.json`:

```json
{
  "theme": {
    "typography": {
      "h1": {
        "fontFamily": "Pretendard",
        "fontSize": "48px",
        "fontWeight": 800,
        "lineHeight": 1.2,
        "letterSpacing": "-0.02em"
      },
      "h2": {
        "fontFamily": "Pretendard",
        "fontSize": "36px",
        "fontWeight": 700,
        "lineHeight": 1.3,
        "letterSpacing": "-0.02em"
      },
      "body": {
        "fontFamily": "Pretendard",
        "fontSize": "16px",
        "fontWeight": 400,
        "lineHeight": 1.5
      },
      "caption": {
        "fontFamily": "Pretendard",
        "fontSize": "14px",
        "fontWeight": 400,
        "lineHeight": 1.4
      }
    }
  }
}
```

Use typography with the `typography` prop:

```tsx
const h1Example = <Text typography="$h1">Heading 1</Text>
const h2Example = <Text typography="$h2">Heading 2</Text>
const bodyExample = <Text typography="$body">Body text</Text>
const captionExample = <Text typography="$caption">Caption text</Text>
```

## Responsive Typography

Define responsive typography using arrays. Each index corresponds to a breakpoint:

```json
{
  "theme": {
    "typography": {
      "h1": [
        {
          "fontFamily": "Pretendard",
          "fontSize": "32px",
          "fontWeight": 800,
          "lineHeight": 1.2
        },
        null,
        null,
        null,
        {
          "fontFamily": "Pretendard",
          "fontSize": "48px",
          "fontWeight": 800,
          "lineHeight": 1.2
        }
      ]
    }
  }
}
```

The breakpoints are:

- Index 0: `0px` (mobile)
- Index 1: `480px`
- Index 2: `768px`
- Index 3: `1024px`
- Index 4: `1280px` (desktop)
- Index 5: `1640px`

Use `null` to skip a breakpoint and inherit from the previous value.

## Using Typography in Components

```tsx
// With Text component
const titleExample = <Text typography="$h1">Main Title</Text>
const bodyExample = <Text typography="$body">Paragraph text goes here.</Text>

// With Box component
const boxExample = <Box typography="$caption">Caption in a box</Box>
```

```tsx
// With styled components
const Title = styled('h1', {
  typography: '$h1',
  color: '$text',
})
```

## Combining with Other Styles

Typography can be combined with other style props:

```tsx
const example = (
  <Text color="$primary" mb={4} textAlign="center" typography="$h1">
    Centered Title
  </Text>
)
```

## Type Safety

Typography tokens are fully type-safe:

```tsx
// TypeScript autocomplete shows: $h1, $h2, $body, etc.
const example = <Text typography="$" />

// Invalid tokens show TypeScript errors
const invalidExample = <Text typography="$invalid" /> // Error!
```
